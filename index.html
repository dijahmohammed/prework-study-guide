HTML

  • The title element defined the title for a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with a webpage.
  • The base element defines the base URL for a webpage.
  • The link element connectd an external resource to the HTML document.
  • The meta element contains information about the page that is used by the browser, such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • The UTF-8 bit or Unicode Transformation Format 8-bit is a character encoding standard utilizing variable-length encoding to efficiently represent a broad range of characters, enabling effective handling of diverse languages and symbols.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations conatin two important components of the css property to apply and add the value of it.
  • In CSS, a class selector is a way to target and style HTML elements based on their class attribute.
  • Selectors are patterns used to select and style HTML elements.
  • The color property is used to set the text color, while background-color sets the background color of an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd command allows for command in Git Bash allows you to change your current working directory to the specified location
  • ls command lists files and folders in current folder
  • pwd displays the "present working directory" path, useful when the full path is not shown in the prompt
  • rm command removes a file

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An "if" statement is used to conditionally execute code based on a specified condition.
  • A "for" loop is used to repeatedly execute a block of code based on a specified condition.
  • An array in JavaScript is a data structure that stores a collection of elements, and it can hold various data types.
